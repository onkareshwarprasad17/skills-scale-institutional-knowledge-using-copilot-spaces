# Role Interaction Matrix

This matrix summarizes primary/secondary responsibilities (RACI-style) for common activities.

| Activity | Release Coordinator | Project Manager | Product Owner | Delivery Lead | DevOps | Stakeholder Liaison |
|---|---:|---:|---:|---:|---:|---:|
| Schedule release | R | A | C | C | C | I |
| Execute release | R | I | I | C | A | I |
| Approve scope change | C | C | A | R | I | I |
| Manage risk mitigation | C | A | I | C | C | I |
| Communicate to stakeholders | I | R | C | I | I | A |

Notes:
- R = Responsible, A = Accountable, C = Consulted, I = Informed
- Use this matrix as a starting point; adapt based on organizational structure.
